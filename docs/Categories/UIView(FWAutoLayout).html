UIView(FWAutoLayout)

@interface UIView (FWAutoLayout)

UIView自动布局分类，兼容UIView和UILayoutGuide(iOS9)

Note

如果约束条件完全相同，会自动更新约束而不是重新添加。 另外，默认布局方式使用LTR，如果需要RTL布局，可通过fwAutoLayoutRTL统一启用

AutoLayout

  • fwAutoLayoutRTL

    是否启用自动布局适配RTL，启用后自动将Left|Right转换为Leading|Trailing，默认NO

    Note

    如果项目兼容阿拉伯语等，需要启用RTL从右向左布局，开启此开关即可，无需修改布局代码 手工切换视图左右布局方法：[UIView appearance].semanticContentAttribute = UISemanticContentAttributeForceRightToLeft;

    Declaration

    Objective-C

    @property (class, nonatomic) BOOL fwAutoLayoutRTL;
  • fwAutoLayout

    是否启用自动布局

    Declaration

    Objective-C

    @property (nonatomic) BOOL fwAutoLayout;
  • +fwAutoLayoutView

    创建自动布局视图

    Declaration

    Objective-C

    + (nonnull instancetype)fwAutoLayoutView;

    Return Value

    自动布局视图

  • -fwAutoLayoutSubviews

    执行子视图自动布局，自动计算子视图尺寸。需先将视图添加到界面(如设置为tableHeaderView)，再调用即可(iOS8+)

    Declaration

    Objective-C

    - (void)fwAutoLayoutSubviews;
  • -fwLayoutHeightWithWidth:

    计算动态布局视图指定宽度时的高度。使用AutoLayout必须约束完整，不使用AutoLayout会调用view的sizeThatFits:方法

    Declaration

    加载器，处理swift不支持load方法问题

    Note

    本方案采用objc扩展方法实现，相对于全局扫描类方案性能高(1/200)，使用简单 使用方法：新增FWLoader扩展objc类方法，以load开头即会自动调用，注意方法名不要重复，建议load+类名+扩展名
